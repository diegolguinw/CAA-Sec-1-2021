Ctedra 3.1: [ Link a la ctedra ]

Se generalizaron los operadores diferenciales a coordenadas curvilneas vistas la semana pasada:

Gradiente: Para $f : \Omega \subseteq \mathbb{R}^3 \rightarrow \mathbb{R}$campo escalar $C^1$:

$$ \nabla f = \frac{1}{h_u}\frac{\partial f}{\partial u}\hat{u} + \frac{1}{h_v}\frac{\partial f}{\partial v}\hat{v} + \frac{1}{h_w}\frac{\partial f}{\partial w}\hat{w}$$

Divergencia: Para $\vec{F} : \Omega \subseteq \mathbb{R}^3 \rightarrow \mathbb{R}^3$campo vectorial $C^1$ :

$$ div(\overrightarrow{F}) = \frac{1}{h_uh_vh_w}(\frac{\partial(F_uh_vh_w)}{\partial u} + \frac{\partial(F_vh_uh_w)}{\partial v} + \frac{\partial(F_wh_uh_v)}{\partial w})$$

Rotor: Para $\vec{F} : \Omega \subseteq \mathbb{R}^3 \rightarrow \mathbb{R}^3$ campo vectorial $C^1$:

$$ rot(\overrightarrow{F}) = \frac{1}{h_uh_vh_w}\left|\begin{matrix} h_u\hat{u} & h_v\hat{v} & h_w\hat{w}\\ \frac{\partial }{\partial u} & \frac{\partial }{\partial v} & \frac{\partial }{\partial w}\\ F_uh_u & F_vh_v & F_wh_w \end{matrix}\right|$$


Ctedra 3.2: [ Link a la ctedra ]

Curva: Una curva es un conjunto $\Gamma \subseteq \mathbb{R} ^3$ que es imagen de un intervalo real $[a,b]$ no vaco a travs de una funcin $\vec{r}: [a,b] \rightarrow \mathbb{R}^3$ continua. A esta funcin la llamaremos la parametrizacin de la curva y en general no es nica.

Se dir que la curva es suave si$\vec{r} \in C^1$, regular si $|| \frac{d\vec{r(t)}}{dt} || > 0, \: \forall t \in (a,b)$, simple si $\vec{r}$ inyectiva y cerrada si $\vec{r}(a) = \vec{r}(b)$.

Observacin: Hay que diferenciar la curva, que es un conjunto, de su parametrizacin, que es una funcin. El matiz que da cada parametrizacin es lo que ser interesante a estudiar, ya que esta puede cambiar el sentido, la velocidad, entre otras cosas.

Longitud de la curva: Se define la longitud de la curva en $t \in [a,b]$ como:

$$s(t) = \int_a^t ||\frac{d\overrightarrow{r}(\tau)}{d\tau}|| \: d\tau $$

N = 100

fig, (ax1, ax2, ax3) = plt.subplots(1,3, figsize=(14,5))

ax1.set_xlim(-0.2, 1.2)
ax1.set_ylim(-0.2, 2.2)

ax2.set_xlim(-0.2, 1.2)
ax2.set_ylim(-0.2, 2.2)

ax3.set_xlim(-0.2, 1.2)
ax3.set_ylim(-0.2, 2.2)

ax1.set_title("Sentido antihorario")
ax2.set_title("Sentido horario")
ax3.set_title("Sentido antihorario, equivalente")

def animate(frame_num):
    if 0 <= frame_num <= N:
        plot = ax1.plot((0,frame_num/N),(0,0), c="purple")
        plot = ax2.plot((0,0),(0,2*frame_num/N), c="orange")
        plot = ax3.plot((0,(frame_num/N)**2),(0,0), c="purple")
        return plot,
    
    elif N+1 <= frame_num <= 3*N:
        plot = ax1.plot((1,1),(0,(frame_num-(N+1))/(N-1/2)), c="purple")
        plot = ax2.plot((0,0.5*(frame_num-(N+1))/(N-1/2)),(2,2), c="orange")
        plot = ax3.plot((1,1),(0,0.5*((frame_num-(N+1))/(N-1/2))**2), c="purple")
        return plot,
    
    elif 3*N+1 <= frame_num <= 4*N:
        plot = ax1.plot((1,1-(frame_num-(3*N+1))/(N-1)),(2,2), c="purple")
        plot = ax2.plot((1,1),(2,2-2*(frame_num-(3*N+1))/(N-1)), c="orange")
        plot = ax3.plot((1,1-((frame_num-(3*N+1))/(N-1))**2),(2,2), c="purple")
        return plot,
    
    elif 4*N+1 <= frame_num <= 6*N:
        plot = ax1.plot((0,0),(2,2-(frame_num-(4*N+1))/(N-1)), c="purple")
        plot = ax2.plot((1,1-0.5*(frame_num-(4*N+1))/(N-1)),(0,0), c="orange")
        plot = ax3.plot((0,0),(2,2-0.5*((frame_num-(4*N+1))/(N-1))**2 ), c="purple")
        return plot,
    
ani = animation.FuncAnimation(fig,func=animate,interval = 20, frames=6*N)
HTML(ani.to_html5_video())
video = ani.to_html5_video()
html = display.HTML(video)
display.display(html)
plt.close()

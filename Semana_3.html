Semana 3: Operadores en coordenadas curvilíneas y curvas¶

Semana del 30/08 al 3/09¶

Cátedra 3.1: [ Link a la cátedra ]

Se generalizaron los operadores diferenciales a coordenadas curvilíneas vistas la semana pasada:

Gradiente: Para $f : \Omega \subseteq \mathbb{R}^3 \rightarrow \mathbb{R}$ campo escalar $C^1$:


$$ \nabla f = \frac{1}{h_u}\frac{\partial f}{\partial u}\hat{u} + \frac{1}{h_v}\frac{\partial f}{\partial v}\hat{v} + \frac{1}{h_w}\frac{\partial f}{\partial w}\hat{w}$$


Divergencia: Para $\vec{F} : \Omega \subseteq \mathbb{R}^3 \rightarrow \mathbb{R}^3$ campo vectorial $C^1$ :


$$ div(\overrightarrow{F}) = \frac{1}{h_uh_vh_w}(\frac{\partial(F_uh_vh_w)}{\partial u} + \frac{\partial(F_vh_uh_w)}{\partial v} + \frac{\partial(F_wh_uh_v)}{\partial w})$$


Rotor: Para $\vec{F} : \Omega \subseteq \mathbb{R}^3 \rightarrow \mathbb{R}^3$ campo vectorial $C^1$:

$$ rot(\overrightarrow{F}) = \frac{1}{h_uh_vh_w}\left|\begin{matrix} h_u\hat{u} & h_v\hat{v} & h_w\hat{w}\\ \frac{\partial }{\partial u} & \frac{\partial }{\partial v} & \frac{\partial }{\partial w}\\ F_uh_u & F_vh_v & F_wh_w \end{matrix}\right|$$


Cátedra 3.2: [ Link a la cátedra ]

Curva: Una curva es un conjunto $\Gamma \subseteq \mathbb{R} ^3$  que es imagen de un intervalo real $[a,b]$ no vacío a través de una función $\vec{r}: [a,b] \rightarrow \mathbb{R}^3$ continua. A esta función la llamaremos la parametrización de la curva y en general no es única.

Se dirá que la curva es suave si $\vec{r} \in C^1$, regular si $|| \frac{d\vec{r(t)}}{dt} || > 0, \: \forall t \in (a,b)$, simple si $\vec{r}$ inyectiva y cerrada si $\vec{r}(a) = \vec{r}(b)$.

Observación: Hay que diferenciar la curva, que es un conjunto, de su parametrización, que es una función. El matiz que da cada parametrización es lo que será interesante a estudiar, ya que esta puede cambiar el sentido, la velocidad, entre otras cosas.

Longitud de la curva: Se define la longitud de la curva en $t \in [a,b]$ como:


$$s(t) = \int_a^t ||\frac{d\overrightarrow{r}(\tau)}{d\tau}|| \: d\tau $$

Parametrización equivalente: Dos parametrizaciones $\vec{r}_1: [a,b] \rightarrow \mathbb{R} ^3$, $\vec{r}_2: [c,d] \rightarrow \mathbb{R} ^3$ de una curva $\Gamma$ se dicen equivalentes si $\exists f: [a,b] \rightarrow [c,d]$ tal que $\vec{r}_1(t) = \vec{r}_2(f(t)), \: \forall t \in [a,b]$.

Curva tipo polígono¶

Parametrizamos un rectángulo compuesto por los vértices $A=(0,0)$, $B=(1,0)$, $C=(1,2)$ y $D=(0,2)$, recorrido en el orden $ABCD$, pero de 3 formas diferentes.

  1. Parametrizamos el rectángulo en sentido antihorario, según la siguiente parametrización:
$$ \vec{r}(t) = (0,t), \: t \in [0,1]$$$$ \vec{r}(t) = (t,1), \: t \in [0,2]$$$$ \vec{r}(t) = (2,1-t), \: t \in [0,1]$$$$ \vec{r}(t) = (2-t,0), \: t \in [0,2]$$
  1. Parametrizamos el rectángulo en sentido horario, según la siguiente parametrización:
$$ \vec{r}(t) = (t,0), \: t \in [0,2]$$$$ \vec{r}(t) = (2,t), \: t \in [0,1]$$$$ \vec{r}(t) = (2-t,1), \: t \in [0,2]$$$$ \vec{r}(t) = (0,1-t), \: t \in [0,1]$$
  1. Parametrizamos el rectángulo en sentido antihorario, pero con una parametrización equivalente a 1, en el que la velocidad ya no es constante.
$$ \vec{r}(t) = (0,t^3), \: t \in [0,1]$$$$ \vec{r}(t) = (\frac{1}{2}t^2,1), \: t \in [0,2]$$$$ \vec{r}(t) = (2,1-t^2), \: t \in [0,1]$$$$ \vec{r}(t) = (2-\frac{1}{2}t^2,0), \: t \in [0,2]$$

N = 50
fig, (ax1, ax2, ax3) = plt.subplots(1,3, figsize=(13,5))

ax1.set_xlim(-0.2, 1.2)
ax1.set_ylim(-0.2, 2.2)

ax2.set_xlim(-0.2, 1.2)
ax2.set_ylim(-0.2, 2.2)

ax3.set_xlim(-0.2, 1.2)
ax3.set_ylim(-0.2, 2.2)

ax1.set_title("Sentido antihorario")
ax2.set_title("Sentido horario")
ax3.set_title("Sentido antihorario, equivalente")

T = np.linspace(0,1,N)

for i in range(1,6):
    T = np.concatenate((T, np.linspace(i,i+1,N)))

def animate(t):
    if 0 <= t <= 1:
        plot = ax1.plot((0,t),(0,0), c="purple")
        plot = ax2.plot((0,0),(0,t), c="orange")
        plot = ax3.plot((0,(t)**2),(0,0), c="purple")
        return plot,
    
    if 1 <= t <= 2:
        plot = ax1.plot((1,1),(0,t-1), c="purple")
        plot = ax2.plot((0,0),(0,t), c="orange")
        plot = ax3.plot((1,1),(0,0.5*(t-1)**2), c="purple")
        return plot, 
    
    if 2 <= t <= 3:
        plot = ax1.plot((1,1),(0,t-1), c="purple")
        plot = ax2.plot((0,t-2),(2,2), c="orange")
        plot = ax3.plot((1,1),(0,0.5*(t-1)**2), c="purple")
        return plot,
    
    if 3 <= t <= 4:
        plot = ax1.plot((1,1-(t-3)),(2,2), c="purple")
        plot = ax2.plot((1,1),(2,2-(t-3)), c="orange")
        plot = ax3.plot((1,1-(t-3)**2),(2,2), c="purple")
        return plot,

    if 4 <= t <= 5:
        plot = ax1.plot((0,0),(2,2-(t-4)), c="purple")
        plot = ax2.plot((1,1),(2,2-(t-3)), c="orange")
        plot = ax3.plot((0,0),(2,2-0.5*(t-4)**2 ), c="purple")
        return plot,
    
    if 5 <= t <= 6:
        plot = ax1.plot((0,0),(2,2-(t-4)), c="purple")
        plot = ax2.plot((1,1-(t-5)),(0,0), c="orange")
        plot = ax3.plot((0,0),(2,2-0.5*(t-4)**2 ), c="purple")
        return plot,
    
ani = animation.FuncAnimation(fig,func=animate,interval = 30, frames=T)
HTML(ani.to_html5_video())
video = ani.to_html5_video()
html = display.HTML(video)
display.display(html)
plt.close()
